Guide

This guide is for the latest version of Kivon. Info in it may or may not function on versions before the latest.

Buttons

Run

The run button will run the commands in the input box and give an output in the output box. If the output says "Error on line...", there is a mistake in your code.

Reset

The reset button will clear the output and input box.

Note: The reset button will also clear the input box! Make sure that you save your code before clicking it.

Home

The home button will bring you to this page.

Version History

The version history will take you to the Version History page.

Report

The report menu will allow you to report a bug or suggest a feature.

Tutorial

The tutorial button will open a small menu at the bottom of the screen which can be used as a step-by-step guide.

Examples

The examples button has 3 examples of Kivon code.

Made By

The made by button gives credit to the creator of Kivon, Kevin George(aka kiwuthegamer).

Share

The share menu allows you share your code.

Choose File

The choose file button will allow you to import a .kivon file from your computer.

Export File

The export file button will export your current Kivon code as a .kivon file.

Load File

After choosing your file, click this button to load it.

Switch Modes

Click this button to open a small menu allowing you to switch the theme of Kivon to multiple themes.

Functions

Print

The print command will instantly type text into the output box.

Syntax - print text

Argument Meaning
text Text to print (Including variables and new lines)

Alert

The alert function will alert some text.

Syntax - alert text

Argument Meaning
text Text to alert (Including variables and new lines)

Prompt

This prompt command will display a prompt box but won’t store the input.

Syntax - prompt text

Argument Meaning
text Text to prompt (Including variables and new lines)

Variable

Normal Variable

The variable command will make a new variable if the variable with the specified name doesnt exist. It will update the current variable if the variable with the specified name does exist.

Syntax - var varname text

Argument Meaning
varname The name of the variable to be set/created
text The text that the variable’s value will be(Including variables and new lines)

Prompt Saved in Variable

This command will save the input of a prompt in a variable.

Note: Other rules of normal variables apply here too.

Syntax - var varname $prompt text

Argument Meaning
varname The name of the variable to be set/created
text The text of the prompt (Including variables and new lines)

Random Number Saved in Variable

This command will save a random number in a variable.

Note: Other rules of normal variables apply here too.

Syntax - var varname $rand num1 num2

Argument Meaning
varname The name of the variable to be set/created
num1 The smallest number that can be generated
num2 The biggest number that can be generated

Calculation In Variable

The calculation in variable will add, subtract, multiply, or divide 2 numbers or variables and set a variable to the result.

Syntax - var varname $calc operator num1 num2

Argument Meaning
varname The name of the variable to be set
operator The operation to be performed on the two numbers
num1 / num2 The numbers or variables to calculate

If

The if statement will compare 2 variables or values with a specified operator. An 'else' can also be added

Syntax - if varname1 operator varname2 then
command
end

Syntax(Else) - if varname1 operator varname2 then
command
else
command
end

Argument Meaning
varname1/varname2 The names of the variables/values to compare
command The command that might run
operator The operator that compares the variables
Operator Meaning
= Is equal to
! Is not equal to

Comment

The comment function allows you to add a comment to your code to help others or to organize it.
Syntax - \\ text

Argument Meaning
text The comment text

Note: A comment will not interact with code. It can only be placed in a separate line.

Function

To create a set of code that can be run repeatedly, use the function command.
Syntax - function function-name
command
end f
Syntax(To call a function) - func function-name times

Argument Meaning
command The command that needs to run
function-name The name of the function
times How many times the function should run
