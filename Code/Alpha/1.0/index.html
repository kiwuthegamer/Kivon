This is the introduction section! Press the buttons on the right and left to move between pages. Click the cross icon to close the tutorial. Each line can have 1 function. There are 3 super simple functions. Let's see the first one.
The print function can be run by typing print as the first word. After typing print, anything you type after it is going to print in the output box.
Example: print Hi
You can even type names starting with $ to print a variable
The other 2 simple functions are alert and prompt. Both of them work the same as the print function, except the first word would be alert or prompt.
The prompt function will not store the reply unless you use a
The variable function is very useful. The variable function will store a value in a variable. You can create and edit a variable with the same command: var varname value
To save the result of a prompt in a variable, using the following command:
var varname $prompt prompt text
To set a variable to a random number use $rand. Example: var varname $rand num1 num2

You can even set a variable to the solution to a math problem. Use $calc: var varname $calc operator num1 num2
The operators for this are "add", "sub", "mul", and "div".
The if statement will allow you to compare 2 values using a condition. If the condition is true then it will run a set of commands. Example:
if varname/value operator varname/value then
commands
end
You can also add an else to the end of an if statement.
if varname/value operator varname/value then
commands
else
commands
end
